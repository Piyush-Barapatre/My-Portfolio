My Projects

Quiz App

The Quiz App is a dynamic web application built using the MERN stack — MongoDB, Express.js, React.js, and Node.js. It allows users to create, take, and manage quizzes in real time. The app includes secure user authentication and authorization using JWT tokens, ensuring data privacy and personalized experiences. MongoDB efficiently handles quiz data, questions, and leaderboard updates, while the React frontend provides an interactive and responsive user interface. I also focused on cross-browser compatibility and smooth UX design to make the app accessible on any device. This project strengthened my skills in full-stack integration, state management, and RESTful API development.

Image Search Engine

The Image Search Engine is a responsive web application that allows users to search and view high-quality images instantly based on entered keywords. Developed using HTML, CSS, and JavaScript, it integrates the Unsplash API to fetch and display relevant images dynamically. The interface is clean, minimal, and optimized for all screen sizes, ensuring a seamless browsing experience across devices. This project enhanced my understanding of working with APIs, asynchronous JavaScript (fetch, promises), and responsive web design techniques.

Captcha Generator

The Captcha Generator is a lightweight JavaScript-based web tool designed to improve website security by preventing automated bot submissions. It generates random combinations of distorted letters and numbers that users must correctly identify before proceeding. Built using HTML, CSS, and JavaScript, the project demonstrates fundamental concepts like DOM manipulation, event handling, and string randomization. The CAPTCHA system plays an essential role in restricting automated interactions such as fake registrations or spam comments, providing an effective human verification mechanism.
